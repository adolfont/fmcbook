# FMCBOOK

## Website

<http://www.tsouanas.org/fmcbook/>

## Instructions

To compile:

    make

To clean up most of the auxiliar files:

    make clean

To clean all auxiliar and built files:

    make cleanall

To upload to the official website (if you have access):

    make upload
